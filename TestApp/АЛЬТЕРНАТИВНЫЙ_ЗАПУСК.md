# Альтернативный способ запуска

Если веб-версия не работает, попробуй:

1. Останови текущий сервер (Ctrl+C в терминале)

2. Запусти обычный Expo сервер:
```bash
cd E:/sait_printer/TestApp
npx expo start
```

3. В открывшемся меню нажми `w` для веб-версии

Или открой в браузере адрес, который покажет Expo (обычно http://localhost:19006)

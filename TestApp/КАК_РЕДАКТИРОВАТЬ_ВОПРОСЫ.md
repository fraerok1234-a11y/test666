# Как редактировать список вопросов

## Способ 1: Прямое редактирование в файле (для небольших изменений)

### Где находятся вопросы:
Файл: `TestApp/constants/tests.ts`

Массив: `QUIZ_QUESTIONS`

### Структура вопроса:
```typescript
{
    id: 1,                              // Уникальный номер вопроса
    question: "Текст вопроса?",         // Текст вопроса
    options: [                          // Варианты ответов (массив из 4 элементов)
        "Вариант 1",
        "Вариант 2",
        "Вариант 3",
        "Вариант 4"
    ],
    correctAnswer: 2,                   // Индекс правильного ответа (0, 1, 2, 3)
    category: "Митне законодавство"     // Категория (опционально)
}
```

### Важно:
- `correctAnswer` - это **индекс** (начинается с 0), а не номер варианта!
  - 0 = первый вариант
  - 1 = второй вариант
  - 2 = третий вариант
  - 3 = четвертый вариант

### Пример редактирования:
```typescript
{
    id: 1,
    question: "Якою є Україна за державним устроєм?",
    options: [
        "федеративною державою",
        "конфедеративною державою",
        "унітарною державою",        // ← Правильный ответ (индекс 2)
        "республікою"
    ],
    correctAnswer: 2,                  // ← Индекс правильного ответа
    category: "Митне законодавство"
}
```

## Способ 2: Редактирование через Word файл (для массовых изменений)

### Шаг 1: Отредактируйте Word файл
Файл: `TestApp/constants/questions.docx`

**Формат в Word:**
- Каждый вопрос на новой строке
- Варианты ответов под вопросом
- **Правильный ответ выделен жирным (Bold)** или другим способом

### Шаг 2: Извлеките данные из Word
```bash
cd TestApp
node extract_docx.js
```

Это создаст файл `extracted_questions.html` с HTML-версией документа.

### Шаг 3: Распарсите вопросы
```bash
node parse_questions.js
```

Это создаст файл `parsed_questions.ts` с массивом вопросов в формате TypeScript.

### Шаг 4: Скопируйте вопросы в tests.ts
Откройте `parsed_questions.ts` и скопируйте массив вопросов в файл `TestApp/constants/tests.ts`, заменив массив `QUIZ_QUESTIONS`.

## Полезные команды

### Посмотреть количество вопросов:
```bash
cd TestApp
grep -c "correctAnswer:" constants/tests.ts
```

### Найти конкретный вопрос:
```bash
cd TestApp
grep -n "текст вопроса" constants/tests.ts
```

### Проверить синтаксис TypeScript:
```bash
cd TestApp
npx tsc --noEmit
```

## Частые ошибки при редактировании

1. **Неправильный индекс correctAnswer**
   - ❌ `correctAnswer: 3` для третьего варианта
   - ✅ `correctAnswer: 2` для третьего варианта (индексация с 0)

2. **Отсутствие запятых**
   - Убедитесь, что каждый элемент массива заканчивается запятой

3. **Неправильные кавычки**
   - Используйте двойные кавычки `"` для строк

4. **Отсутствие закрывающих скобок**
   - Каждый вопрос должен быть объектом `{ ... }`

---

## Рекомендации

- Для **небольших изменений** (1-10 вопросов): используйте Способ 1
- Для **массовых изменений** (много вопросов): используйте Способ 2
- **Всегда делайте резервную копию** файла `tests.ts` перед большими изменениями
- После редактирования **проверьте приложение** - запустите тест и убедитесь, что все работает

# Инструкция по запуску программы

## Как запустить программу с GitHub репозитория

### Вариант 1: Если у вас ещё нет кода на компьютере

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/fraerok1234-a11y/testss.git
   cd testss
   ```

2. **Установите зависимости:**
   ```bash
   npm install
   ```
   или
   ```bash
   yarn install
   ```

3. **Запустите программу в режиме разработки:**
   ```bash
   npm run dev
   ```
   или
   ```bash
   yarn dev
   ```

4. **Откройте браузер:**
   - Перейдите по адресу: http://localhost:3000
   - Программа будет автоматически обновляться при изменении файлов

### Вариант 2: Если код уже есть на вашем компьютере

1. **Перейдите в папку проекта:**
   ```bash
   cd e:\sait_printer
   ```

2. **Убедитесь, что зависимости установлены:**
   ```bash
   npm install
   ```

3. **Запустите программу:**
   ```bash
   npm run dev
   ```

4. **Откройте браузер:**
   - http://localhost:3000

## Другие команды

- **Сборка для продакшена:**
  ```bash
  npm run build
  ```

- **Запуск продакшен версии:**
  ```bash
  npm run build
  npm run start
  ```

- **Проверка кода (линтинг):**
  ```bash
  npm run lint
  ```

## Требования

- Node.js (версия 18 или выше)
- npm или yarn

## Проблемы?

Если возникают ошибки:
1. Убедитесь, что Node.js установлен: `node --version`
2. Удалите папку `node_modules` и файл `package-lock.json`, затем выполните `npm install` заново
3. Проверьте, что порт 3000 не занят другим приложением

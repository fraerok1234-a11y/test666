# Как открыть программу на iPhone (iOS)

## Способ 1: Через Wi-Fi сеть (самый простой)

### Требования:
- ✅ iPhone и компьютер должны быть подключены к **одной Wi-Fi сети**
- ✅ Сервер должен быть запущен с параметром `-H 0.0.0.0`

### Шаги:

1. **Узнайте IP адрес вашего компьютера:**
   - На Windows: откройте командную строку и введите `ipconfig`
   - Найдите "IPv4-адрес" (например: `192.168.31.100`)

2. **Запустите программу:**
   ```bash
   npm run dev
   ```

3. **На iPhone откройте Safari и введите:**
   ```
   http://192.168.31.100:3000
   ```
   ⚠️ **Важно:** Замените `192.168.31.100` на IP адрес вашего компьютера!

4. **Если не открывается:**
   - Проверьте, что iPhone и компьютер в одной Wi-Fi сети
   - Убедитесь, что брандмауэр Windows не блокирует порт 3000
   - Попробуйте отключить VPN на iPhone, если он включен

---

## Способ 2: Через туннель (ngrok) - работает из любой сети

Если телефон и компьютер в разных сетях, используйте туннель:

1. **Установите ngrok:**
   - Скачайте с https://ngrok.com/download
   - Или через npm: `npm install -g ngrok`

2. **Запустите программу:**
   ```bash
   npm run dev
   ```

3. **В другом терминале запустите ngrok:**
   ```bash
   ngrok http 3000
   ```

4. **Скопируйте URL из ngrok** (например: `https://abc123.ngrok.io`)

5. **Откройте этот URL на iPhone в Safari**

---

## Способ 3: Развернуть на хостинге (Vercel) - постоянный доступ

### Через Vercel (рекомендуется для Next.js):

1. **Загрузите код на GitHub** (уже сделано ✅)

2. **Перейдите на https://vercel.com**

3. **Войдите через GitHub**

4. **Нажмите "New Project"**

5. **Выберите репозиторий `fraerok1234-a11y/testss`**

6. **Нажмите "Deploy"**

7. **Через 1-2 минуты получите URL** (например: `https://testss.vercel.app`)

8. **Откройте этот URL на iPhone** - будет работать из любой сети!

---

## Решение проблем

### Брандмауэр блокирует подключение:

1. Откройте "Брандмауэр Защитника Windows"
2. Нажмите "Разрешить взаимодействие с приложением"
3. Найдите "Node.js" и разрешите для частных сетей

### Не открывается на iPhone:

- Убедитесь, что используете `http://` (не `https://`)
- Проверьте, что порт 3000 указан: `:3000`
- Попробуйте перезапустить сервер
- Проверьте, что iPhone и компьютер в одной сети
